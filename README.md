# Movie-Review-Sentiment-Analysis

This project performs classification on IMDb movie review dataset using BERT and RoBERTa. 
The aim of the project is to compare the result of the two algorithms using the based version of both models

Below is the dataset reference 

Large Movie Review Dataset: https://ai.stanford.edu/~amaas/data/sentiment/
